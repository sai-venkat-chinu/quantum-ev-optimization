# Quantum-Inspired EV Charging Optimization

## Overview
This project demonstrates a quantum-inspired approach to optimizing EV charging decisions using quantum computing.

## Objective
To simulate multiple EV charging combinations and identify optimal charging patterns.

## Methodology
- Used Qiskit for quantum simulation
- Represented EVs as qubits
- Generated multiple charging states using quantum superposition
- Measured outcomes to find optimal solutions

## Results
The histogram shows different EV charging combinations. Higher frequency states indicate better optimization.

## Tools Used
- Python
- Qiskit
- Google Colab
